Empty-cell format (no instructions). Colored = characterized, dark = unused/zero.

Scaling rules (empty lane):
Payload length = 0x1000 × (⌊(rows+1)/2⌋ + 1) — e.g. 1 row = 0x2000, 4 rows = 0x3000, 9 rows = 0x6000
Header row word = (rows + 1) × 0x20 — stored LE at entry 0, offsets +0x00/+0x01
When empty: grid rows at 0x0A60 + row × 0x0800 — 32 columns × 64 bytes each
